# Vibe Tag Rules

定義 `tools/vibe_summarizer.py` 中 `DEFAULT_TAG_RULES` 所使用的關鍵字對應表。
非工程師隊友可直接修改本文件來新增/調整標籤，再同步更新 Python 檔案。

| Tag | 觸發關鍵字 |
|-----|-----------|
| `#出片` | 好拍、拍照、打卡、網美、景觀、裝潢、氛圍、文青、復古、美照 |
| `#排隊` | 排隊、候位、等位、客滿、熱門、爆滿、預約、人超多 |
| `#老字號` | 老字號、古早味、在地、傳統、開了、祖傳、多年 |
| `#CP值` | 平價、便宜、划算、cp值、大份、高cp |
| `#辣感` | 辣、重口味、麻辣 |
| `#豆腐鍋專門` | 豆腐鍋、嫩豆腐、順豆腐、순두부、soondubu |

## 新增標籤方式

1. 在本文件 table 加一行
2. 在 `tools/vibe_summarizer.py` 的 `DEFAULT_TAG_RULES` 新增對應 `TagRule`
