## Pipeline 參數（重跑用）

| Demo 檔名 | `--query` | `--logic` |
|-----------|-----------|-----------|
| `功能1-挑嘴朋友聚餐` | 朋友聚餐，日式或台式，步行25分鐘內 | 排除牛肉料理，辣度可調或無辣，有蔬食選項但非素食專門店，有質感不無聊 |
| `功能2-有酒有飯餐酒館` | 餐酒館或熱炒，步行25分鐘內 | 可以點酒、不要太貴、份量要吃飽、台式熱炒或小酒館風格 |
| `功能3-預算有限但要體面` | 請客但預算有限，台式，步行25分鐘內 | 看起來體面、不要一看就很便宜、實際價格中等合理 |
| `功能4-可以常去不需訂位` | 可以常去的店，韓式，步行25分鐘內 | 不用訂位、不會太吵、餐點有品質但不會吃膩、適合單獨用餐或小聚 |
| `情境1-久別重逢長聊` | 朋友聚餐，可以聊很久，義式，步行25分鐘內 | 久坐不趕人、安靜、朋友不吃牛我想吃，桌距寬鬆 |
| `情境2-帶外國朋友私房台味` | 帶外國朋友吃台灣味，步行25分鐘內 | 有台灣特色但不觀光客爆多、台灣人也會推薦的私房店、食材道地 |
| `情境3-帶長輩早鳥晚餐` | 帶長輩吃晚餐，和食，步行25分鐘內 | 五點多就開門、不太舊但不太潮、不會太吵、適合長輩口味 |
| `情境4-曖昧初次約飯` | 曖昧對象第一次單獨吃飯，泰式，步行25分鐘內 | 不太浪漫避免尷尬但也不像朋友聚餐、有設計感但不是純網美店、餐點有記憶點不要太貴 |

## 重跑 demo

若 demo 結果過期，用以下指令重新預跑（會覆蓋舊檔）：

```bash
uv run python -m tools.cli agent \
  --query "<query>" \
  --user-lat 25.047094 --user-lng 121.542698 \
  --logic "<logic>" \
  --top-k 3 \
  --save-demo "<demo 檔名>"
```
