# 測試集

Demo case 已預跑並存入 `.demo/`。Foodie Agent 開場選單的 D1–D8 對應如下。

## 功能

| # | 情境描述 | Demo 檔名 |
|---|---------|-----------|
| 1 | 跟一群很挑嘴的朋友聚餐，有人不吃牛、有人不吃辣、有人只吃健康餐，但又不想吃那種無聊的健康便當店 | `功能1-挑嘴朋友聚餐` |
| 2 | 可以點酒但不要太貴的餐酒館或熱炒店，餐點要可以吃飽 | `功能2-有酒有飯餐酒館` |
| 3 | 要請客但預算有限，希望看起來體面一點；不要那種一看就很便宜的地方，但實際價格不能太高 | `功能3-預算有限但要體面` |
| 4 | 想找一間可以常去的店，不用訂位，也不會太吵；餐點有一定品質但不會吃膩 | `功能4-可以常去不需訂位` |

## 情境

| # | 情境描述 | Demo 檔名 |
|---|---------|-----------|
| 1 | 跟很久不見的朋友見面，想找可以聊很久的店，他不吃牛，但我想吃 | `情境1-久別重逢長聊` |
| 2 | 想找有特色的店帶外國朋友去，但不要觀光客爆多的地方；最好是台灣人也會推薦的私房名單 | `情境2-帶外國朋友私房台味` |
| 3 | 帶長輩吃晚餐，最好五點多就開的店，不要太舊的店但也不要太潮或太吵 | `情境3-帶長輩早鳥晚餐` |
| 4 | 跟曖昧對象第一次單獨吃飯，不想太浪漫避免尷尬，但也不能像朋友聚餐；希望環境有設計感但不要網美店，餐點要有記憶點但不要太貴 | `情境4-曖昧初次約飯` |

## 重跑 demo

若 demo 結果過期，用以下指令重新預跑（會覆蓋舊檔）：

```bash
uv run python -m tools.cli agent \
  --query "<query>" \
  --user-lat 25.047094 --user-lng 121.542698 \
  --logic "<logic>" \
  --top-k 3 --no-safe-mode \
  --save-demo "<demo 檔名>"
```
